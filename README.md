# Main game operators
- Launcher.java is responsable for launching the game.
- WindowDisplay.java is responsable for creating the game window.
- Screen.java is responsable for rendering objects, creating objects, updating all objects and checking collisions with all objects.
- Background.java is responsable for the color of the background of the game.

# Player
- Player.java is responsable for making the skeleton and the main functionalities of the player.
- Bullet.java is responsable for making the skeleton and functionality for a bullet.

# Enemy
- Enemy.java is responsable for making the skeleton and functionality for an enemy.
- EnemyTrail.java makes the enemies have a trail.

# Powerups
- Immortal.java and Speed.java are responsable for giving the player upgrades, i.e. speed and immortality.
